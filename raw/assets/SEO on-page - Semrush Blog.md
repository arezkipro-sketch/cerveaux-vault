### Guide d’initiation aux meta tags pour le SEO

Les meta tags fournissent des informations sur une page web et influencent directement la façon dont les moteurs de recherche évaluent un site web. Dans ce guide d’initiation, vous apprendrez à optimiser votre SEO on-page grâce aux meta tags. Vous lirez également des conseils sur les meilleurs outils à utiliser. Nous nous pencherons sur les balises title, bien sûr, mais aussi sur les balises méta robots et les balises méta viewport.
