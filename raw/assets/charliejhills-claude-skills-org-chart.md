# 42 Claude Skills organisées comme une vraie entreprise

Source : @charliejhills — 304k vues

## Comment installer un skill Claude Code

1. Aller sur le lien GitHub du skill
2. Dans Claude Code, taper `/install-skill <url-du-repo>`
   — ou —
   Copier le fichier SKILL.md du repo dans `~/.claude/skills/<nom-du-skill>/`
3. Le skill est disponible immédiatement via `/nom-du-skill`

Pour les skills sur `claude.com/plugins` : les ouvrir directement dans le navigateur et cliquer "Install".
