# Création de site web — Vue d'ensemble

Méthode pour créer et vendre des sites web à des commerces locaux via Claude Code, en utilisant Google Maps comme outil de prospection et un système d'outreach basé sur la démo pré-construite.

---

## Le loop complet

```
FIND      → Google Maps (4 filtres) → liste de 20 prospects par niche
BUILD     → Claude Code + frontend-design + UI/UX Pro Max → démo en ~1 soirée
SHIP      → Hébergement ($40-45/an) seulement après le oui
SELL      → Message froid : vidéo 40s + 1 problème précis + prix fixe
RÉCURRENT → Care plan $50/mois → revenu mensuel passif
```

---

## 1 — Prospection Google Maps

### Les 4 filtres
1. Niche où un site génère du CA (restaurants, dentistes, gyms, med spas, artisans, immobilier)
2. Note **4.4+ avec 50+ avis** → preuve de trésorerie
3. Site daté, cassé sur mobile, ou absent
4. Ville assez grande pour avoir 20+ candidats dans la niche

### Tableur de prospection
| Colonne | Contenu |
|---|---|
| Nom du business | — |
| Contact | Email propriétaire ou Instagram |
| **Problème précis** | Ce qui ne va pas avec leur site actuel ← devient l'ouverture du message |

---

## 2 — Build avec Claude Code

### Setup initial (une fois)
- [ ] Claude Pro ($20/mois) → app desktop → mode code
- [ ] Installer `frontend-design` (Anthropic) et `UI/UX Pro Max` (communauté)
- [ ] Activer le mode **auto** (Claude agit sans demander permission)
- [ ] Récolter 3-5 screenshots de référence (Dribbble / Awwwards / Pinterest)

### Workflow
```
1. Prompt de build /ui-ux-pro-max + références visuelles
2. Claude pose 5-7 questions de clarification → répondre avec précision
3. Première version en ~10 min
4. Pass 1 : grader (typo / couleur / hiérarchie / motion / mobile / copy)
5. Pass 2 : rendre cher (batch de micro-interactions, pas une par une)
6. Pass mobile dédié
7. Enregistrer vidéo 40s de scroll (desktop + mobile)
```

**Règle :** une démo par niche, pas une par business.
**Ne pas héberger avant le oui.**

---

## 3 — Outreach

```
Sujet : made this for [Business Name]

Hi [Name], I redesign websites for [niche] in [city].
I already built a concept for [Business Name]. 40-second video attached.

Your current site [problème précis].
This one fixes that.

If you like it, it's live on your domain this week.
Flat $[price], no subscriptions, no agency retainer.
If not, no hard feelings, the video's yours to keep.
```

**Règles :**
- Vidéo en avant, pas les credentials
- 1 problème précis (colonne du tableur)
- Prix fixe, clair
- Ne jamais dire "AI-built"
- Follow-up 3 jours après → double le taux de réponse

## 4 — Pricing

| Prestation | Prix |
|---|---|
| One-pager petit marché | $500-800 |
| One-pager métropole | $1 000-2 500 |
| Care plan mensuel | **$50/mois** |
| Claude Pro | $20/mois |
| Hébergement + domaine 1 an | $45 |
| **Marge** | **~90%+** |

10 clients care plan = $500/mois récurrents.

---

## Ramp réaliste
20 envois → 3-6 réponses → 0-2 deals.
2 clients/mois à $1 500 = **$3 000/mois à temps partiel**.

---

## Les 5 erreurs fatales
1. Polir une démo une semaine au lieu d'envoyer (envoyer à 90%)
2. Pitcher la technologie ("site AI") au lieu de l'outcome
3. Choisir des niches qu'on aime au lieu des niches qui paient
4. Pas de follow-up après le premier message
5. Pricer à $200 — signal de camelote, attire les pires clients
