## Checklist des Principes de Base du SEO

Pour garantir le succès de votre SEO, vous devez en maîtriser les bases. Si vous ne le faites pas, votre site risque d'avoir du mal à se classer.

Les conseils suivants sont des tâches SEO de base, mais elles constituent les fondements d'une stratégie SEO réussie.

### 1\. Configurez Google Search Console et Bing Webmaster Tools

[Google Search Console (GSC)](https://search.google.com/search-console/) est un outil qui vous aide à surveiller, maintenir et corriger les performances de votre site dans les résultats de recherche de Google.

C'est l'un des meilleurs outils pour mieux comprendre comment Google voit votre site web.

![Google Search Console, rapport sur les performances](https://static.semrush.com/blog/uploads/media/6b/44/6b44add9cc51b9fe11b41b68759a1389/8efff70bf25a99f6b0e08be7aa5c7f5e/original.png)

### 6\. Vérifiez les Actions Manuelles dans la Search Console

Une tentative manifeste de violer ou de manipuler [Directives de Google pour les webmasters](https://developers.google.com/search/docs/essentials?hl=fr) entraîne des actions manuelles.

Si votre site fait l'objet d'une action manuelle, une partie ou la totalité de votre site n'apparaîtra pas dans les résultats de recherche de Google.

Les actions manuelles visent le spam généré par l'utilisateur, les problèmes de données structurées, les liens non naturels (à la fois vers et depuis votre site), le contenu superficiel, le texte caché et bien d'autres choses encore.

Il est conseillé de vérifier si des actions manuelles ont été effectuées dans [Google Search Console](https://search.google.com/search-console/about).

Pour cela, faites défiler la barre de navigation latérale de l'outil jusqu'à ce que vous trouviez la section de menu " **Sécurité et actions manuelles** ". Cliquez dessus. Cliquez ensuite sur " **Actions manuelles** ".

![Actions manuelles dans Google Search Console](https://static.semrush.com/blog/uploads/media/85/8c/858cee14ca072de39c1931bbc57417ad/9f43842006521a232d991a74e188dc7a/original.jpeg)

Actions manuelles dans Google Search Console

Cela devrait permettre d'obtenir un rapport sur les problèmes détectés. S'il y en a.

Google vous informera dans un [rapport GSC](https://support.google.com/webmasters/answer/9044175?hl=fr) si votre site a fait l'objet d'une action manuelle.

Mais si vous travaillez sur un nouveau projet ou reprenez un site, les actions manuelles devraient être l'une des premières choses à vérifier.

### 12\. Analysez l'Intention des Pages qui se Classent

[L'intention de recherche](https://fr.semrush.com/blog/analyse-intention-de-recherche/) est la raison pour laquelle un utilisateur effectue une recherche.

Il existe quatre types d'intentions de recherche:

- **Navigationnelle** : Essayer de trouver quelque chose
- **Informationnelle** : Essayer d'en savoir plus sur quelque chose
- **Transactionnelle**: Essayer de réaliser une action spécifique
- **Commerciale** : Essayer d'en savoir plus avant de prendre une décision d'achat

L'un des principaux objectifs de Google est de faire en sorte que le contenu de votre page corresponde à l'intention de recherche des internautes.

Veillez donc à ce que le contenu de votre page corresponde toujours à l'intention de l'internaute pour le mot clé que vous ciblez.

Pour trouver rapidement l'intention d'un mot clé, utilisez notre outil [Vue d'ensemble des mots clés](https://fr.semrush.com/analytics/keywordoverview/).

Entrez votre mot clé, appuyez sur **Recherche**, et vous verrez un cadre intitulé "Intention".

### 16\. Vérifiez la Présence de Versions Dupliquées de votre Site dans l'Index de Google

Il est important que vous n'autorisiez Google à indexer qu'une seule version de votre site.

De même, vos visiteurs ne doivent pouvoir accéder à votre site qu'à un seul endroit. Dans le cas contraire, cela peut entraîner des problèmes d'exploration, d'indexation et de sécurité

Par exemple, les sites suivants sont tous des versions différentes de votre site. Mais elles doivent toutes pointer vers une seule.

`https://www.domaine.com   https://domaine.com   http://www.domaine.com   https://domaine.com`

C'est à vous de décider si vous choisissez une version non www ou www. Mais il est préférable d'opter pour l'une ou l'autre.

Toutes les autres versions doivent faire l'objet d'une [redirection 301](https://fr.semrush.com/blog/seo-basic-redirections-301-quand-comment/) vers la version principale.

Vous pouvez rapidement le vérifier en saisissant chaque variante dans la barre de votre navigateur. Si vous êtes redirigé vers la version principale de votre site, tout devrait être en ordre.

### 22\. Utilisez une Structure d'URL SEO-Friendly

Les structures d'URL SEO-friendly permettent aux moteurs de recherche de parcourir plus facilement vos pages. Et de comprendre de quoi elles parlent ou quelle est la fonction qu'elles remplissent.

Les URL de vos pages doivent être simples et descriptives. Pour les moteurs de recherche comme pour les utilisateurs.

Voici à quoi ressemble une URL adaptée au SEO:

`https://www.domaine.com/chaussures-rouges/`

En revanche, voici une chaîne de requête qui n'est pas descriptive:

`https://www.domaine.com/categorie.php?id=32`

**Utilisez** des traits d'union dans vos URL pour séparer les mots. **N'utilisez pas** de tirets bas.

Veillez à ce que les URL soient aussi courtes que possible. [Une étude de Backlinko](https://backlinko.com/search-engine-ranking) a montré que les URL plus courtes ont tendance à être mieux classées.

Par exemple, vous pouvez utiliser des données structurées pour les recettes, les événements, les organisations, les livres, les entreprises locales et les avis.

### 34\. Trouvez et Résolvez les problèmes de Cannibalisation de Mots Clés

On parle de cannibalisation des mots clés lorsqu'une ou plusieurs pages de votre site ciblent le même mot clé et la même intention.

L'une des deux pages affecte la capacité de l'autre à se classer. Aucune des deux pages ne sera aussi performante qu'elle pourrait l'être.

Vous aurez du mal à vous classer pour ces mots clés parce que les moteurs de recherche auront des difficultés à déterminer quelle page afficher dans les SERP.

Pour savoir si votre site présente des problèmes de cannibalisation de mots clés, utilisez l'outil [Suivi de position](https://fr.semrush.com/position-tracking/).
