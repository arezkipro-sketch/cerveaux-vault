# Apprenez à booster vos vidéos avec l'algorithme YouTube — Summary

**One-liner:** How YouTube's ML recommendation algorithm decides what each user sees (Search, recommendations, Trending, Shorts) in 2025, and the creator strategies that "seduce" it — all aimed at maximizing total watch time on the platform.

## Key takeaways
- The algorithm is a **personalized ML curator** deciding which videos appear per user across **Search, the recommendations feed, the Trending tab, and Shorts**. Its objective: **maximize total time spent on the platform**.
- **Core ranking signals:** watch time, **CTR** (thumbnails + titles), engagement (comments/likes/shares/subs), **audience retention** (watched to the end), and user history/personalization.
- **Search ranking** blends content relevance + channel authority: keywords in title/description/tags, user search history, popularity/engagement, and **recency**.
- **Recommendations feed** predicts the *next* video: similarity to already-watched, behavior of similar-profile users, engagement velocity, and temporal/viral trends.
- **Trending tab** rewards **speed and consistency of engagement**, not absolute views: fast view-growth, engagement relative to channel size (small channels can break through), local relevance, format diversity.
- **Shorts (2025)** rewards rapid repeated interactions, retention, **rewatches**, and immediate engagement; attractive thumbnails, popular sounds, and auto-captions boost reach.

## Notable strategies (with locations)
- Create **video series** → cumulative watch time + return visits, read by the algorithm as a relevance signal *(§Créer des séries)*.
- Add **subtitles/captions** → accessibility + reach + a relevance signal the algorithm uses to understand the topic *(§Ajouter des sous-titres)*.
- Hook viewers in the **first few seconds**; production quality (light/sound/edit/narration) drives retention *(§contenu pertinent)*.
- Win **Trending** via intense engagement in the **first hours** + accelerated view growth *(§Maximiser… Tendances)*.
