# Rapport Liens de GSC — Summary

**One-liner:** GSC's free "Links" report shows external backlinks + internal links — top linked pages, top linking sites, top anchor text — the free way to analyse your link profile.

## Key takeaways
- **Data points:** most-linked pages, top linking sites, top link (anchor) text, top internally-linked pages.
- **Why:** backlinks are a confirmed ranking factor (a "vote of confidence"); the report reveals your strongest pages + link sources to build on.
