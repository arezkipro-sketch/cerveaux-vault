# Guide du débutant en référencement naturel (SEO)

Guide officiel France Num (Direction générale des Entreprises), rédigé par Rémi Brandini (Consultant Google, Activateur France Num). Destiné aux TPE/PME qui débutent en SEO. Mis à jour le 24 octobre 2025.

**Valeur spécifique :** source institutionnelle française (donc crédible pour le contexte franco-français), utilise la terminologie SEO francophone standard — notamment "netlinking" / "tissage de liens" pour les backlinks — et cadre le SEO en **3 leviers** (vs le modèle à 4 piliers Semrush).

## Overview

Le SEO est présenté comme accessible aux TPE/PME sans budget conséquent. L'objectif : corriger les problématiques les plus fréquentes lors d'audits SEO, en autonomie.

## Key Takeaways

1. **SEO = 3 leviers** (terminologie francophone officielle) :
   - **Technique** : structuration correcte des contenus (balises title, Hn, listes, vitesse, données structurées)
   - **Mots clés** : sélection et intégration naturelle dans les contenus
   - **Popularité** : netlinking — obtenir des liens depuis d'autres sites
2. **Netlinking** (= link building / backlinks) : terme français dominant ; stratégie basée sur partenariats, citations, annuaires de qualité. Qualité > quantité. Éviter l'achat de liens.
3. **Schema.org e-commerce** : balises `Product`, `Offer`, `Review`, `AggregateRating` pour afficher avis et prix dans les SERP.
4. **Google Business Profile** = outil prioritaire pour les activités locales (pas de données structurées custom nécessaires).
5. **Contenu original** : ne jamais dupliquer — les moteurs valorisent l'unicité. L'expert SEO vaut pour les cas complexes.

## Detailed Notes

### Levier 1 — Technique

**Balise title :**
- Descriptive, 50-60 chars, mot-clé important en début
- Accrocheur pour maximiser le CTR dans les SERP

**Balises Hn :**
- H1 unique = titre principal de la page
- H2 = sous-sections principales
- H3 = détails de sous-sections
- Inclure mots-clés pertinents dans les Hn

**Balises liste :**
- `<ul>` (puces) et `<ol>` (numérotées) → lisibilité + identification des points clés par les bots

**Vitesse de chargement + images :**
- Compression images (outils en ligne gratuits)
- Formats : JPG / PNG / **WebP** (meilleure compression sans perte visible)
- Redimensionner aux dimensions d'affichage réelles
- Attribut **ALT** : description pertinente + mot-clé → référencement dans Google Images
- Mise en cache des images (navigateur) → accélère visites récurrentes

**Données structurées (Schema.org) :**

| Type | Balises utilisées | Usage |
|---|---|---|
| Produit | `Product`, `Offer` | Nom, prix, disponibilité dans les SERP |
| Avis | `Review`, `AggregateRating` | Étoiles et commentaires clients dans les SERP |
| Local | Google Business Profile | Plus simple et efficace que les données structurées maison |

### Levier 2 — Mots clés

- Outils payants : Semrush, Ahrefs
- Outils gratuits (suffisants pour débutant) : Ubersuggest, Keywordtool.io
- Méthode simple : tester les mots-clés directement dans Google → volume de résultats + format dominant → indication de pertinence
- Intégration naturelle : titres, intro, corps, Hn, meta description
- **Meta description** : 50-160 chars, pas de facteur de classement direct, influence le CTR

### Levier 3 — Popularité (Netlinking)

- **Netlinking** = terme français pour link building / backlinks
- 3 tactiques recommandées pour TPE/PME :
  1. **Partenariats stratégiques** — contenu invité, collaborations, recommandations croisées (sites pertinents et de confiance uniquement)
  2. **Références et citations** — publier études, rapports, statistiques originales → d'autres sites vous citent naturellement
  3. **Annuaires d'entreprises** — répertoires locaux ou de niche de qualité (pas les annuaires spam)
- Règle absolue : **qualité > quantité**
- Pratiques interdites : achat de liens, réseaux artificiels → pénalité Google

### Analyse des performances

- Outils d'analyse : GA4 (trafic + comportement), GSC (positionnement)
- Suivre : pages à fort potentiel, lacunes, tendances du secteur
- Conversions = proportion de visiteurs réalisant une action cible (vente, inscription, devis)

### Conseil expert

- Faire du SEO en interne est possible pour les bases
- Consultant SEO = utile pour : audit complet, problématiques complexes, gain de temps, veille des tendances
