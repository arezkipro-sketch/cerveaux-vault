# Santé du site web — Summary

**One-liner:** Site health = the foundation letting a site perform; a composite of security, usability and search-optimisation metrics, measured via a site audit.

## Key takeaways
- **What:** a set of metrics scoring a site on security, usability, and search visibility; a healthy site → more traffic + conversions.
- **Audit components:** crawlability, on-page SEO, technical SEO (structure, mobile), mobile-friendliness, speed.
- Tools weight components differently; produces an overall health score.
