- Based on context from [[jotaro-seo-content-strategy]] (published later), key image SEO rules include:
  - ALT attribute: descriptive, contains keyword (not "IMG_4821.jpg" or "photo produit").
  - Format: WebP — same quality, 2-3× smaller file size.
  - Weight: max 100–150 KB per image; 1 MB+ = serious performance problem.
  - File naming: descriptive kebab-case (e.g. "console-retrogaming-portable.webp").
  - Google Images = undervalued acquisition channel.
