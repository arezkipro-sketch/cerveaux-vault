# Soumettre un sitemap à Google — Summary

**One-liner:** An XML sitemap lists a site's pages for crawlers; submit it via the GSC "Sitemaps" report so Google finds and indexes pages faster.

## Key takeaways
- **XML sitemap** = file listing pages for search engines to discover/index. URL usually `domain.com/sitemap.xml` or `/sitemap_index.xml` (find via `sitemap site:domain.com filetype:xml`).
- No sitemap? Generate one with a sitemap-generator tool.
- **Submit:** connect site to [[google-search-console]], then add the sitemap URL in the Sitemaps report — Google can find it alone, but submitting ensures discovery + faster ranking.
