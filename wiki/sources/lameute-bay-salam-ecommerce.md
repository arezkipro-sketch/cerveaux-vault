# Bay' Salam en e-commerce — La Meute (PDF)

Ebook de La Meute. Guide complet et pratique sur la conformité islamique du dropshipping via le contrat Bay' Salam. Source principale sur ce sujet dans le vault — première source sur le domaine "e-commerce islamique".

**Valeur spécifique :** explique pourquoi le dropshipping classique est haram (3 causes : gharar, riba, absence de damân), présente le Bay' Salam comme solution licite avec ses 5 conditions, fournit un exemple concret de fiche produit Shopify conforme, des clauses CGV copiables, et liste les erreurs qui annulent le contrat.

## Thèse centrale

Le dropshipping classique est haram car le vendeur vend ce qu'il ne possède pas sans cadre contractuel clair → introduction de gharar, risque de ribâ (bay' ad-dayn bi ad-dayn), et absence de damân.

Le Bay' Salam est l'exception islamique légitimée par le Prophète ﷺ qui permet la vente à livraison différée sous conditions strictes → rend le dropshipping licite si appliqué correctement.

## Structure de l'ebook

1. Introduction : modèles e-commerce et interdiction islamique
2. Dropshipping : définition + avantages
3. Pourquoi le dropshipping classique est haram : hadith + consensus + 3 causes
4. Bay' Salam : définition + législation Coran/Sunna + différence vente interdite + 5 conditions
5. Application pratique : parcours client + exemple fiche Shopify + clauses CGV
6. Erreurs qui annulent le contrat (3 erreurs critiques)
7. Points de vigilance du commerçant musulman (Jumu'a + tromperie marketing)
