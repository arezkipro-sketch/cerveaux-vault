# Copywriting vs Rédaction web — Summary

**One-liner:** Two complementary disciplines — copywriting converts, web writing attracts/informs via SEO — best combined.

## Key takeaways
- **Copywriting** = persuasive text to provoke an action (buy, signup, quote); leans on persuasion, storytelling, psychology.
- **Rédaction web** = SEO-optimised, pleasant-to-read content to attract organic traffic (blog posts, info pages, product sheets).
- **Objectives:** copywriting → convert reader into client/prospect; web writing → inform, educate, improve search visibility.
- **Techniques diverge:** copy = catchy headlines, benefits-over-features, repeated CTAs, psychological levers. Web writing = keyword/tag optimisation, internal linking, clear hierarchy, factual completeness, online-readable style.
- **They combine:** an SEO blog post can embed copywriting to convert (newsletter signup, purchase); a sales page can be SEO-optimised to rank.

## Notable claims (with locations)
- Example: e-commerce selling courses → web writing on "Comment apprendre le copywriting sans expérience" attracts traffic; copywriting on the sales page converts. — *(§Exemple)*
