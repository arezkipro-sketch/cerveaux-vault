# Script vidéo persuasif — Summary

**One-liner:** Most videos don't convert because the *script* is weak, not the editing; a 6-part structure (the "90-second" template) fixes it.

## Key takeaways
- **5 goals:** captivate fast, connect ("c'est moi"), install one simple central idea, prove, trigger action.
- **Pre-work:** one action + one problem + one benefit (never 4 objectives); know audience + use the mirror effect.
- **6-part structure (timed):** Hook (0-3s) → Identification (3-10s) → Central idea (10-20s) → Demonstration/proof (20-60s) → Transition to offer (60-75s) → CTA (75-90s).
- **7 hook types:** direct question, objection, shock phrase, clear promise, contrast, mistake, curiosity.
- **Central idea** = one memorable repeatable sentence ("la clarté vend plus que la créativité").
- **Channel adaptation:** Reels/TikTok (aggressive hook, fast, 1 idea), YouTube (narrative), sales page (proof/objections), ad (ultra-short).
- **Mistakes:** talk about product not viewer, no structure, too many ideas, no proof, weak CTA.
