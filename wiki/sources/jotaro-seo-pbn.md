# PBN (Private Blog Network) — Summary

**One-liner:** A PBN is a network of privately-owned sites (often built on expired domains) that each link to a "site mère" to artificially inflate its authority — a powerful but Google-penalized link-building strategy.

## Key takeaways
- **Definition**: a Private Blog Network (PBN) = a set of interconnected sites all pointing via backlinks to one "site mère" (target site), with the goal of consolidating authority and TrustRank on the target.
- **Mechanism**: each PBN satellite site passes [[link-juice]] to the site mère, boosting its domain authority and SERP positions.
- **Implementation steps**: (1) define target site + thematic, (2) find expired domain names in the same niche with good authority history, (3) check domain history (no spam associations), (4) build or buy sites on these expired domains, (5) publish thematically consistent content, (6) create backlinks to site mère.
- **Thematic consistency is mandatory**: a backlink from a car rental site to a sneaker site is worthless. The entire PBN network must be on the same theme as the target.
- **Risk**: Google explicitly penalizes PBNs when detected. Manual or algorithmic penalty = deindexation of target site.
- **Use case**: e-commerce sites wanting to boost their site mère quickly. Author presents this as educational but implicitly warns of risks.

## Notable claims (with locations)
- Expired domains (NDD expirés) preferred over new domains because they carry existing authority — *(step 3)*.
- PBN extends TrustRank as well as domain authority — *(tweet 2.1)*.
