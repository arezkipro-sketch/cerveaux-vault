# Topic clusters — Summary

**One-liner:** A topic cluster = a pillar page + interlinked sub-pages on related sub-topics; covering a subject deeply builds topical authority and ranks more keywords.

## Key takeaways
- **Structure:** one **pillar page** (broad overview) + multiple **cluster pages** (sub-topics), all tied by [[maillage-interne|internal links]] (hub-and-spoke).
- **Why:** more quality content on a topic → rank more keywords + build topical authority; signals E-E-A-T to Google.
- Example: pillar "decorate an apartment" + clusters on kitchen/living-room/bedding.
