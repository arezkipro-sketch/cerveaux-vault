# Erreurs copywriting (contenu web) — Summary

**One-liner:** Common web-content copy mistakes — ignoring the audience, fuzzy message, skipping social proof — plus readability and test/optimise discipline.

## Key takeaways
- **3 core mistakes:** neglecting the target audience (HubSpot: audience-tailored content = +200% ROI); fuzzy message (one key message, no jargon: "réduit votre temps de réponse de 30%" > "procédés optimisés"); skipping social proof (BrightLocal: 79% trust online reviews like personal recs).
- **Write well:** catchy titles (numbers/questions/power words), readability (short sentences, subheads, bullets, tables), test & optimise (CTR, time-on-page, A/B titles/CTAs).
- **Prep:** research audience + build personas, analyse competitors, plan structure first.
