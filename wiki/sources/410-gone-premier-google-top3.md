# Comment être premier sur Google : viser le Top 3 — Summary

**One-liner:** « Être premier » n'a pas un seul sens (payant / local / organique / position zéro) ; on gagne le Top 3 organique en cochant **trois cases dans l'ordre — pertinence → qualité → autorité** — via une méthode en 5 étapes.

## Key takeaways
- **« Première place » est polysémique** : selon la requête, la SERP est occupée par Ads, pack local (MyGoogleBusiness), PAA, vidéos, position zéro, organique classique. Chaque snippet a son propre algorithme → toujours **analyser la SERP du mot-clé ciblé** d'abord.
- **Dominer la page 1 > être juste premier** : viser tous les formats (vidéo, PAA, shorts, organique) = autant de visibilité → [[search-everywhere-optimization]].
- **Triptyque de sélection du n°1** → [[facteurs-de-classement]] :
  1. **Pertinence** (filtre n°1) — la page répond EXACTEMENT à l'intention + sous-intentions ; comparer aux 3 mieux classés.
  2. **Qualité** — utile, unique, clair, crédible, scannable ; E-E-A-T ; « mérite-t-elle vraiment la 1ère place ? »
  3. **Autorité** — départage à pertinence/qualité égales ; pèse d'autant plus que la requête est concurrentielle.
- **Méthode 5 étapes** :
  1. **Gagner l'intention** — « respirer l'odeur de la SERP » (Laurent Bourelly) : mot-clé + synonymes + sous-intentions longue traîne + types de contenus positionnés + angle + axe de différenciation. Test : montrer le plan (H1/H2/H3) à un proche → comprend-il le sujet sans le mot-clé ?
  2. **Contenu crédible (E-E-A-T en pratique)** — montré DANS la page (vécu, précision, sources, transparence), pas juste une bio ; auto-évaluation scoring E-E-A-T → [[e-e-a-t]].
  3. **On-page** — title/meta marketing (promesse+bénéfice+angle), intention dans les 10 premières lignes, 1 H1 + H2 = sous-questions SERP/PAA, formats extractibles, maillage, preuves → [[seo-copywriting]].
  4. **Technique & UX** — images légères, mobile-first, indexation (canonicals propres, sitemap, pas de noindex), Core Web Vitals, 404/500 propres → [[core-web-vitals]], [[mobile-seo]], [[http-status-codes-seo]].
  5. **Autorité & backlinks** — d'abord **autorité interne** (liens depuis pages fortes, ancres variées, cohérence cluster) puis backlinks qualité/pertinence/régularité → [[maillage-interne]], [[backlinks]].
- **Pyramide inversée** + « page qui fait gagner du temps » = principe de qualité (réponse d'abord, du général au spécifique).
- **Backlinks propres** : RP digitales, guest posts qualitatifs, partenariats, récupération de mentions, **linkable assets** (checklist/étude/outil/template). Éviter annuaires douteux/réseaux artificiels.
- **Requêtes concurrentielles** : l'autorité pèse beaucoup plus ; progresser par paliers **top 10 → top 5 → top 3 → top 1** ; découper en sous-requêtes + cluster.
- **SEO vs SEA** : SEO = actif durable ; SEA = visibilité immédiate (tant que budget). Ads = accélérer/tester/débloquer ; le meilleur scénario = **Ads nourrit le SEO** (identifier les requêtes qui convertissent → priorités SEO) → [[sem]].

## Notable claims (with locations)
- **CTR par position 2025** (étude Growthsrc, post-IA Overview, requêtes anglophones) : pos 1 organique = **19 %** des clics ; **top 3 ≈ 43 %** ; positions 5-9 quasi identiques ; **position 10 rebondit** (plus visible) → [[click-through-rate]].
- IA Overview : **CTR des premières places en baisse**, CTR des dernières positions de page 1 en hausse → [[ai-overviews]].
- E-commerce / contexte : 38 % des prospects utilisent désormais ChatGPT pour trouver des prestataires → [[ai-seo]].
- Citation Google : systèmes de classement priorisent « informations utiles et fiables, people-first » *(developers.google.com)*.
