# SEO mobile — Summary

**One-liner:** Optimising a site to rank and read well on mobile; ~62% of global web traffic is mobile (2025), so it's essential, not optional.

## Key takeaways
- **Why:** ~62% of traffic is mobile; ~90% of consumers use phones to research products in-store.
- **Responsive design is Google's recommended setup:** one URL + HTML, CSS adapts per device. Benefits: single URL (easy share/links), less maintenance, no redirect (faster), Googlebot crawls once, no duplicate content.
- Ties to mobile-first indexing (Google indexes the mobile version).
