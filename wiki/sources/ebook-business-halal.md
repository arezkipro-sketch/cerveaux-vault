# Ebook Business Halal

**One-liner :** Guide complet 70 pages en 2 parties — Adam (business, logistique, sourcing Alibaba, agent chinois) + Mimine (SEO Shopify complet) — pour créer un e-commerce halal de A à Z.

## Overview

Ebook collaboratif "Business Halal" combinant setup business (partie Adam, pages 1-23) et SEO Shopify détaillé (partie Mimine, pages 24-70). Co-auteurs identifiés : "Adam" pour la partie business/logistique, et Mimine = même personne que [[guide-dropshipping-seo-halal]] et [[mimine-aliexpress-fournisseurs]].

⚠️ **Conflits avec d'autres sources :**
- **Pinterest** : ce guide dit 5-15 épingles/jour vs [[guide-dropshipping-seo-halal]] qui dit 20-30/jour
- **Linkuma** : plateforme de backlinks recommandée avec lien affilié Mimine → biais commercial

## Key Takeaways

1. **Budget minimum 1 000-2 000 €** pour démarrer (stock + site + acquisition).
2. **Progression logique** : produits génériques via agent chinois → ODM → OEM sur best-sellers.
3. **3 modèles logistiques** : domicile / 3PL / agent chinois. Agent = meilleur pour démarrer vite avec peu.
4. **Niche volume seuil = 28-32k recherches/mois transactionnelles** (home + collections cumulées, hors produits + blogs).
5. **EORI obligatoire** pour importer hors UE — gratuit, sur douane.fr.
6. **DDP = incoterm idéal débutant** : tout inclus dans le prix de livraison (douane, taxes).
7. **Alibaba : fabricants, pas trading companies** + verified + trade assurance.
8. **10 critères agent chinois** avant de signer (ancienneté, entrepôt, MOQ, app Shopify...).
9. **30/70 règle de paiement** : 30% pré-production, 70% après contrôle qualité validé.
10. **Méthode PPA** = écrire un blog par question PAA (max 700 mots, pyramide inversée) — 10 blogs/jour possible.

## Partie 1 — Business Setup (Adam)

### Halal e-commerce — interdits fondamentaux

- Pas de femmes ni d'awra d'hommes dans les visuels
- Pas de musique
- Pas de paiement fractionné avec frais (vérifier la conformité)
- Pas de vente sans stock (Bay' Salam)
- Pas de contrefaçons
- Pas de publicité mensongère

### Budget de démarrage

**Minimum recommandé : 1 000-2 000€** tout compris (stock, site, acquisition trafic, marketing). Réinvestir les bénéfices pour scaler.

### 3 modèles e-commerce

| Modèle | Avantage | Inconvénient |
|---|---|---|
| **Produits génériques** | Rapide, MOQ faible, coût bas | Tout le monde vend la même chose |
| **ODM (marque blanche)** | Différenciation logo/packaging | +investissement, lead time +long |
| **OEM (produit propre)** | Différenciation maximale, avantage durable | +coûteux, +risqué, lead time très long |

**Stratégie débutant recommandée** :
1. Génériques via agent chinois → tester le marché rapidement et à moindre coût
2. Si marché valide → passer en ODM via agent
3. Puis ODM via stock en France (meilleurs délais)
4. Finalement OEM sur les best-sellers

### 3 modèles logistiques

| Modèle | Pour qui | Avantages | Limites |
|---|---|---|---|
| **Domicile** | Débutant, petit budget | Contrôle total, meilleurs délais | Chronophage, gestion manuelle |
| **3PL** | Business qui tourne | Délégation complète | Payant, moins adapté au démarrage |
| **Agent chinois** | Lancement rapide, hijra | Capital faible, pas de gestion logistique | Délais plus longs, risque qualité agent |

> ⚠️ Agent déconseillé pour produits lourds/volumineux (livraison trop chère).

### Transport Chine → France

- **Bateau** : moins cher, plus long → à privilégier en stock de base
- **Avion** : cher, rapide → urgences uniquement
- **Stratégie 70/30** : 70% par bateau + 30% par avion pour éviter les ruptures

### Setup légal pré-lancement

**Statut recommandé débutant : Micro-entreprise**
- Gratuit à créer, pas de TVA (sous seuil)
- Plafond CA : 188 700€ pour vente de biens (jusqu'en 2025)
- ACRE disponible pour cotisations réduites
- CFE : exonéré l'année de création

**Comptes à créer avant la boutique :**
- Stripe (processeur de paiement)
- PayPal Pro
- Gmail dédié (puis Google Workspace si croissance)
- Revolut (compte séparé pro/perso — strict)

**EORI** : numéro obligatoire pour importer hors UE → gratuit sur le site de la douane française.

### Sourcing Alibaba

**Filtres à activer** : "Verified" + "Alibaba Trade Assurance"

**Critères fournisseur** :
- Minimum 2-3 ans d'ancienneté
- Fabricant (pas trading company)
- Bonnes performances dans le profil entreprise
- Certifications et normes vérifiées (contacter la douane ou DGCCRF pour identifier les normes obligatoires)

**Template email Alibaba** (en anglais) : introduction + description précise + images du produit + 6 questions (samples, tarifs EXW/DDP 500/1000/2000 unités, certifications, lead time, moyens de paiement, WeChat/WhatsApp).

**Règle de paiement** :
- Petites commandes : Alibaba Trade Assurance + PayPal Pro
- Grosses commandes : 30% pré-production + 70% après contrôle qualité validé (jamais 100% d'un coup)

**Contrôle qualité** : payer des entreprises d'inspection (SGS, V-Trust, Bureau Veritas) avant de payer le solde.

**Transporteur Alibaba** : rechercher "freight forwarder DDP France" → critères : verified + trade assurance + 4-5 ans + 35-40% exports Europe + basé Guangdong/Zhejiang.

### 10 critères pour choisir un agent chinois

1. Société (plus sécurisé) vs indépendant
2. Ancienneté ≥2-3 ans
3. Entrepôt dans la même région que le fournisseur + près d'un aéroport
4. Spécialisation stockage/envoi (pas trop de services divers)
5. Service principal clairement défini
6. Traitement <48h + livraison <10 jours (DDP avion)
7. Coûts détaillés : stockage + livraison + tous frais annexes
8. Habitué au marché français / Europe de l'Ouest
9. MOQ faible accepté (démarrage sans commandes)
10. Application Shopify intégrée (automatisation des commandes) ← fortement recommandé

**Paiement agent** : mensuel/bimensuel via Alibaba trade assurance + PayPal Pro → puis virement si confiance.

## Partie 2 — SEO (Mimine)

*(Beaucoup d'éléments recouvrent les sources Jotaro déjà dans le vault. Seuls les points différenciants ou complémentaires sont listés ici.)*

### Validation de niche — Volume seuil précis

**Seuil minimum : 28 000-32 000 recherches/mois transactionnelles** (homepage + mots-clés collections cumulés — hors produits et blogs).

Calculer avec Semrush "Vue d'ensemble des mots-clés" → additionner les volumes transactionnels.

> **Différence vs Jotaro** : Jotaro parle de KD <30% en phase 1 — ici le critère est le volume minimum niche, pas la difficulté.

### Niche — 5ème technique méconnue

**Shopify SHOP** (shop.app) — marketplace officielle Shopify disponible aux USA. Chercher une niche qui marche aux US et l'importer en France (si elle répond aux 4 critères).

### SpeedFly — Astuce prix

Thème SpeedFly à 197€ (1 licence = 3 sites). **Astuce** : ajouter au panier, remplir les infos perso sans finaliser le paiement, attendre 1 jour → recevoir un code promo de 30€.

### Analyse concurrentielle — Ratios à calculer

- **Ratio backlinks/domaines référents** : beaucoup de backlinks pour peu de domaines = profil off-site mauvais
- **Authority Score** : <20 = faible, 20-30 = moyen, >30 = sérieux (toujours relatif à la niche)
- Trafic concurrent vs volume niche : si concurrent fait 25k/mois sur niche 35k = très bien optimisé

### Marge produit — calcul complet

**Marge nette cible : 25-30% minimum** (après imposition).

Coûts à inclure : produit + importation + flacon/étiquette + packaging + carton + papier kraft + carte de remerciement + livraison client + frais Shopify/processeur + imposition (sur CA, pas bénéfice en micro-entreprise).

**Micro-entreprise** : imposition sur CA → `Profit net = CA - imposition - toutes dépenses`.

### 4 axes de différenciation

1. SEO meilleur que les concurrents
2. Branding fort (nom, logo, site)
3. ODM/OEM le plus tôt possible
4. Marketing site (promos, up-selling, cross-selling, code promo vs email)

### Blog — Méthode PPA (People Also Ask)

**Méthode PPA** = format sous-estimé, très efficace pour monter rapidement.
- Chercher mot-clé sur Google → noter toutes les questions PAA → créer 1 blog par question
- Chaque blog : max 700 mots, pyramide inversée (répondre dès la première ligne)
- Rythme : 10+ blogs/jour possible
- Ajouter photos, liens vers 1-2 collections max, H2 + H3, mots-clés en gras
- Tips blog : lien vers sites d'autorité (Wikipedia, sites gouvernement) — Google apprécie

> **Complémentaire à [[jotaro-seo-content-strategy]]** (1 article/jour pilier) — la méthode PPA est plus rapide mais produit des articles courts.

### Netlinking — Linkuma

Auteur recommande **Linkuma** (app.linkuma.com) : marketplace backlinks à partir de 50€/mois.
- Plans : Starter 7€ HT, Linkuma 10€ HT, Boost 30€ HT, Marketplace au prix du site
- ⚠️ Lien affilié Mimine dans l'ebook → biais commercial
- Alternative blanche : guest blogging, partenariats, contenu de qualité pour liens naturels

> Vs Jotaro : Jotaro recommande 120-250€/backlink (DR 25-40). Linkuma propose des liens à partir de 7€ — qualité moindre probable.

### Pinterest Organic — ⚠️ Conflit volumes

| Paramètre | Cet ebook | [[guide-dropshipping-seo-halal]] |
|---|---|---|
| Épingles/jour | **5-15** | **20-30** |
| 1 board = | 1 collection | 1 collection |
| Descriptions | Uniques (pas de duplication) | Uniques |
| Métriques à suivre | Conversions, impressions, clics, enregistrements, CTR | — |

**Données Pinterest** : 454 millions d'utilisateurs actifs/mois, 60% de femmes, croissance +9%/an.

**Meilleures niches Pinterest** : décoration (intérieur/extérieur), vêtements, jardinage, mode.

**Process en 9 étapes** :
1. Choisir images (devant + derrière, toutes variantes couleurs)
2. Créer compte professionnel
3. Réutiliser les mots-clés du site
4. Nommer boards avec mots-clés des collections
5. Descriptions uniques (pas de copier-coller des fiches)
6. Optimiser le site pour Pinterest
7. Extension "Bouton Enregistrer Pinterest" (6M utilisateurs, 3,5★)
8. Garder profil actif
9. Connecter compte Pinterest à Shopify

### Autorité Google — Page "Qui sommes-nous"

Créer une page "Qui sommes-nous" sur le site = signal d'autorité E-E-A-T. Raconter l'histoire de la marque. Couplé à présence réseaux sociaux active → évite le "shadow ban" Google.

### Mega Menu — Ordre par volume

- Collections principales : classer de gauche à droite par volume de recherche décroissant
- Sous-collections : classer de haut en bas par volume de recherche décroissant

*(Exception : boutique islamique → peut prioriser l'ordre pédagogique sur l'ordre SEO)*

- Compresser les images
