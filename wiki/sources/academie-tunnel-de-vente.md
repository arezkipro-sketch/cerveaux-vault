# Tunnel de vente — Summary

**One-liner:** A structured path turning a stranger → qualified prospect → client in 3-5 stages (TOFU/MOFU/BOFU); copywriting is what makes each stage convert.

## Key takeaways
- **Definition:** funnel = the journey from first brand contact to purchase; vs a vitrine site (informs), a funnel orients/reassures/pushes to action. Each visitor is at a different maturity stage.
- **3 stages:** **TOFU** (attract — SEO, social, ads, educational content + lead magnet to capture email) → **MOFU** (nurture — email sequences, case studies, testimonials, webinars) → **BOFU** (convert — sales page, offer, guarantee, CTA).
- **6-step build:** define audience + main problem → create an offer that solves it → capture page (one title, one benefit, one button) → email sequence (5-7 emails over 10-14 days) → sales page (AIDA/PAS) → optimise checkout (fewer clicks, reassurance, no forced account).
- **Sourced stats:** single-CTA pages convert ~2.5× better (HubSpot); users read ~20% of page text (Nielsen Norman); ~73% of leads aren't ready to buy on first contact (MarketingSherpa); ~97% leave without acting.
- **Mistakes:** too many steps, generic message, no follow-up, ignoring data. Iterate via A/B testing.

## Notable claims (with locations)
- "68% des entreprises n'ont pas optimisé leur tunnel." — *(résumé)*
- Worked example: copywriting-freelance course funnel (blog → free guide → 5 emails → sales page → checkout). — *(§exemple)*
