# Query fan-out experiment — Summary

**One-liner:** A real experiment optimising for "query fan-out" — Google AI Mode decomposing a query into many sub-queries — to test if covering sub-questions improves AI visibility.

## Key takeaways
- **Query fan-out:** AI Mode breaks one query into multiple sub-queries, retrieves per sub-query, then synthesises — so covering the sub-questions on a page may improve odds of being cited.
- Most AI-optimisation advice is speculative; this is an actual experiment to see what works.
