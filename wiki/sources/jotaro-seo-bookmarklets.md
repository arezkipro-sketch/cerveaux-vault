# SEO Bookmarklets — Summary

**One-liner:** Browser bookmarklets (JavaScript snippets saved as browser favorites) automate repetitive SEO inspection tasks like `site:` commands with a single click.

## Key takeaways
- **Bookmarklet**: a JavaScript snippet saved as a browser bookmark. Clicking it on any page triggers the script.
- **Installation**: right-click browser bookmarks bar → Add page → Name = action name, URL = the bookmarklet JavaScript code.
- **Example bookmarklet — `site:` command**: opens a new tab with `site:[current-domain]` search, showing all indexed pages of the site. Useful for rapid indexation checks.
- Thread lists multiple bookmarklets but only `site:` is fully captured in the clip.

## Concepts introduced or updated
- No major new concepts. Tactical tooling.

## Connections / contradictions
- `site:` command connects to [[google-bots]] (what's indexed) and [[crawl-budget]] (which pages Google chose to index).
