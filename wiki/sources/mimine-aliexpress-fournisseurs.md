# Comment trouver des fournisseurs sur Aliexpress

**One-liner :** Guide opérationnel de 10 pages (même auteur que [[guide-dropshipping-seo-halal]]) — extensions Chrome, filtres de recherche AliExpress, 9 critères fournisseur, 8 critères produit, et le piège "choice".

## Overview

Guide pratique sans fioritures sur la qualification de fournisseurs AliExpress pour le dropshipping. Contenu 100% opérationnel : quels filtres activer, quels critères vérifier, comment éviter les pièges. Auteur = mimine_95 (Twitter/TikTok), même auteur que [[guide-dropshipping-seo-halal]].

Distinction clé posée dès l'intro : **AliExpress = dropshipping** / **Alibaba = marque propre**. Ces deux cas ne se mélangent pas.

## Key Takeaways

1. **4 extensions Chrome indispensables** pour travailler sur AliExpress (voir section dédiée).
2. **Seuil fiabilité fournisseur = 95% d'avis positifs** + ≥10 000 abonnés + ancienneté ≥1 an.
3. **Règle fondamentale** : si le fournisseur est fiable → tous ses produits sont prenables, même à 0 commandes. Si le fournisseur n'est pas fiable → prendre uniquement le produit ciblé, sans s'abonner.
4. **Piège "choice"** : le badge "choice" sans page boutique complète = collection AliExpress interne, pas un vrai fournisseur → pas de soutien en cas de problème.
5. **Politique de retour = piège majeur** : la garantie "satisfait ou remboursé" AliExpress = souvent remboursement partiel. Prévoir un budget retours dès le départ.
6. **Logo "Plus"** = meilleur signal qualité fournisseur (livraison ≤10 jours garantie + retour gratuit + livraison gratuite).

## Extensions Chrome Recommandées

| Extension | Fonction principale | Utilisateurs |
|---|---|---|
| **AliTools** | Fiabilité vendeur multi-critères | 4 000 |
| **Asify** | Pays cible, démographie acheteurs, dl vidéos/images | 50 000 |
| **AliSave Plus** | Téléchargement images/vidéos en 1 clic (meilleure qualité) | 100 000 |
| **AliExpress Search by Image** | Recherche par image → trouver fournisseur alternatif | 200 000 |

> AliExpress Search by Image est particulièrement utile quand un best-seller est en rupture : clic droit → retrouver le même produit chez un autre fournisseur en 2 clics.

## Filtres de Recherche Initiaux

1. **Trier par** : Commandes (les plus vendus en premier)
2. **Note minimum** : 4 étoiles ou plus
3. **Langue** : Français
4. **Pays** : France (si vente en France)
5. **Livraison** : Livraison Rapide activée (≤10 jours depuis Chine)
6. **Prix min** : 50€+ si produits high ticket

> Disclaimer auteur : parfois les filtres combinés bugguent sur AliExpress → zéro résultat. Tester sans tous les filtres si nécessaire.

## Checklist Fournisseur (9 critères)

| Critère | Seuil / Règle |
|---|---|
| **Avis positifs** | ≥95% (cliquer le % pour voir le détail) |
| **Délais livraison** | ≤10-12 jours, filtre "Livraison Rapide" |
| **Ancienneté** | ≥1 an d'activité sur AliExpress |
| **Volume commandes** | Vérifier les produits "best-sellers" de sa boutique |
| **Abonnés boutique** | ≥10 000 abonnés |
| **Prix** | Juste milieu — trop bas = qualité faible, trop haut = marges serrées |
| **Politique retour** | ⚠️ Souvent remboursement partiel — prévoir budget |
| **Réactivité** | Envoyer un message test → réponse ≤24h |
| **Logo qualité** | "Plus" = meilleur signe (livraison garantie + retour gratuit) |

## Checklist Produit (8 critères)

| Critère | Seuil |
|---|---|
| **Avis** | Avec images (plus fiables) |
| **Délai livraison** | ≤12 jours |
| **Frais livraison** | <5€ |
| **Expédition** | ≤48h par le fournisseur |
| **Note** | ≥4,5 étoiles |
| **Commandes** | ≥300 si fournisseur douteux · 0 OK si fournisseur fiable |
| **Images** | Qualité suffisante pour la boutique |
| **Retours** | Gratuits |

## Règle Fournisseur vs Produit

```
Fournisseur fiable ?
  OUI → prendre tous ses produits (même à 0 commandes)
  NON + produit OK → prendre uniquement l'URL de ce produit
            ↳ ne pas s'abonner, ne pas prendre d'autres produits de lui
```

## Piège "Choice" AliExpress

Un badge "choice" sans page boutique complète (pas d'avis %, pas de date ouverture, pas d'abonnés) = **collection interne AliExpress**, pas un vrai fournisseur. En cas de problème → aucun support. À éviter.

## Comment Trouver des Produits

1. Recherche par mot-clé niche → filtrer "choice" / "plus" / "top selling"
2. Sur la fiche produit : section **"ARTICLES SIMILAIRES"** → autres bons produits
3. Plus bas : section **"VOUS AIMEREZ AUSSI"** → autres découvertes
4. Extension Asify : démographie acheteurs + pays qui convertit
