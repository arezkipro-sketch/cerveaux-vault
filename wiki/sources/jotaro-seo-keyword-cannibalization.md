# Keyword Cannibalization — Summary

**One-liner:** Keyword cannibalization occurs when multiple site pages target the same keyword, creating internal competition that dilutes authority and hurts SERP positions.

## Key takeaways
- **Definition:** Multiple pages targeting the same keyword → internal competition → Google confused about which page to show → neither ranks well.
- **Detection methods**: (1) list all keywords and check which articles rank for each, (2) search the keyword on Google and see which site pages appear, (3) check positions for duplicate-keyword pages, (4) use Google Search Console (click a keyword → Pages tab to see all pages ranking for it — comment by @Lamaxaw).
- **Solutions**: (1) strong [[maillage-interne]] / crosslinking, (2) [[cocon-semantique]] to organize pages into thematic groups with clear hierarchy.
- [[cocon-semantique]] explicitly cited as the architectural solution to cannibalization.
