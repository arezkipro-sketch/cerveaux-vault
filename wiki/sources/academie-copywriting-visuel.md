# Copywriting visuel — Summary

**One-liner:** Pairing words with images, colour and layout to capture attention and guide to action — text alone no longer suffices.

## Key takeaways
- **Why:** brain processes an image ~60,000× faster than text *(stat, unsourced)*; visuals boost memorisation + engagement.
- **4 fundamentals:** visual hierarchy (titles/subheads/bullets/contrast guide the eye) · image choice (authentic, supports an argument, not generic stock) · layout (spacing/alignment/responsive → lower bounce) · graphic consistency (colours/fonts/style = trust).
- **Applications:** landing pages (hook title + demonstrative image + contrasted CTA), emails (header image + short text + button), blog (illustrations/infographics).
- **Mistakes:** generic images, visual overload, unoptimised image weight (hurts SEO), non-responsive.
- **× SEO:** alt tags + light images, hierarchical HTML, better UX/retention = positive Google signals.
