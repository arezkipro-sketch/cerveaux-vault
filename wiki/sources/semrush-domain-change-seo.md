# Changement de nom de domaine & SEO — Summary

**One-liner:** Changing your domain changes every page's URL — backlinks to the old domain stop pointing to you, so it can feel like starting from SEO scratch; mitigate with redirects.

## Key takeaways
- **Impact:** affects authority, rankings, traffic; main risk = backlinks to the old domain no longer point to the new one. Exact impact unpredictable, but loss is not inevitable.
- **When to change** (rebrand etc.) vs when not — weigh carefully; preserve equity with 301s + GSC change-of-address + updating internal links.
