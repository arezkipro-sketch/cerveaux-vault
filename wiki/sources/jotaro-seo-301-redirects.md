# 301 Redirects in SEO — Summary

**One-liner:** 301 redirect signals a permanent page move to search engines, preserving link juice and traffic — essential when restructuring URLs, merging pages, or changing domains.

## Key takeaways
- **Definition:** HTTP 301 = permanent redirect. Signals to Google that a page has moved permanently to a new URL.
- **[[link-juice]] preservation**: redirecting old URL → new URL transfers SEO authority. Without it, inbound links lose their value (page drops, traffic gone).
- **When to use**: site refactoring, URL structure changes, page mergers, domain migration.
- **Avoid overusing**: each 301 loses a small amount of link juice. Redirect chain = cumulative loss. Don't redirect blindly.
- **User experience benefit**: visitors are automatically sent to the new page; no frustrating 404.
