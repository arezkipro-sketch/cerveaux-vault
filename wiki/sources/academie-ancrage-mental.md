# Ancrage mental — Summary

**One-liner:** The first info the reader sees becomes the reference point that frames everything after (value, price, credibility); if you don't set the anchor, the prospect sets an unfavourable one.

## Key takeaways
- **Anchoring:** first information = an anchor; the rest is judged in comparison. ("2000€ then 700€" makes 700€ reasonable.) You change *perception*, not reality.
- **Rule #1:** if you don't set the anchor, the prospect does — usually badly ("just text", "expensive for a freelance").
- **5 anchor types:** (1) contrast (before/after) (2) categorisation/repositioning ("un texte" → "un actif de conversion") (3) cost of inaction (what the problem costs) (4) proof (validated anchor) (5) framing (same reality, more useful angle: "une dépense" → "un investissement").
- **Sales-page sequence:** set a strong anchor at the open → anchor on transformation not features → reinforce with proof → **reveal price only after framing value**. Order: **valeur → preuves → mécanisme → prix**.
- **5 mistakes:** anchor too far from reality, too many anchors (confusion), anchor without proof, price too early (becomes the main anchor), vague framing.
