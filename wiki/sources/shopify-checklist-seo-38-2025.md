# Checklist SEO complète : 38 conseils pour 2025 — Summary

**One-liner :** Checklist Shopify en 38 points répartis sur 6 domaines (bases, on-page, contenu, technique, off-page, local) + 4 bonus 2025 — orientée e-commerce débutant/intermédiaire, avec biais Shopify explicite.

## Overview

Article publié par Shopify FR en novembre 2024, ciblant les propriétaires de boutiques en ligne. Couvre tous les piliers SEO dans une progression logique setup → contenu → technique → liens → local → tendances. Utile comme grille de vérification opérationnelle, moins dense que [[semrush-seo-checklist-41]] mais plus accessible et e-commerce-first.

## Key Takeaways

1. **Setup obligatoire avant tout** : GSC + Bing Webmaster + GA4 + sitemap soumis + indexation vérifiée (`site:domain.com`). Les boutiques Shopify en essai gratuit sont indexées, mais le travail disparaît si on ne passe pas au plan payant.
2. **On-page fondamentaux** : 1 H1 contenant le mot clé ; title ≤ 60 caractères avec mot clé en début ; meta description ≤ 155 caractères ; URL courtes avec tirets ; balises alt descriptives sur toutes les images.
3. **Contenu e-commerce** : ne pas copier les descriptions fabricant (duplicate content) ; blogging = preuve d'expertise thématique → signaux E-E-A-T → hausse de classement ; pages de destination dédiées par sujet > page d'accueil générique.
4. **Technique** : mobile-first (test Google), HTTPS obligatoire, PageSpeed Insights pour CWV, corriger les liens cassés via GSC, maillage interne (navigation + articles croisés) + texte d'ancrage varié (exact / partiel / connexe / marque / titre).
5. **Off-page** : analyser les backlinks concurrents → comprendre pourquoi ils obtiennent des liens ; guest posts + mentions presse + récupération de mentions sans lien → backlinks à faible friction.
6. **Local SEO** : Google Business Profile = indispensable pour les commerces physiques ; cohérence NAP sur tous les annuaires ; page dédiée par ville si multi-emplacements.
7. **Bonus 2025** : l'IA (Shopify Magic, ChatGPT) accélère la production mais le contenu générique IA sera pénalisé ; diversifier les formats (vidéo, images, infographies) selon ce que Google récompense dans les SERP cibles ; inviter des experts tiers pour renforcer l'E-E-A-T ; toujours optimiser l'UX (vitesse, mobile, tunnel de commande).

## Detailed Notes

### Section 1 — Checklist de base (6 points)

| # | Tâche |
|---|---|
| 1 | Configurer Google Search Console |
| 2 | Configurer Bing Webmaster Tools |
| 3 | Soumettre un sitemap (Shopify : `/sitemap.xml` auto-généré) |
| 4 | Configurer Google Analytics (GA4) |
| 5 | Choisir des outils SEO (Moz, Ahrefs, Semrush, GSC natif) |
| 6 | Vérifier que le site est indexé (`site:domaine.com`) |

### Section 2 — Checklist on-page (8 points)

| # | Tâche | Règle |
|---|---|---|
| 7 | Recherche de mots-clés | Volume + intention + longue traîne ; 1 page = 1 intention |
| 8 | Balise H1 | 1 seule par page, contient le mot-clé principal |
| 9 | Balise title | ≤ 60 caractères, mot-clé en début, lisible humainement |
| 10 | Meta description | ≤ 155 caractères, convaincante, mot-clé inclus |
| 11-12 | URL | Lisibles, tirets (pas tirets bas), courtes, mot-clé cible |
| 13 | Balises alt images | Descriptives, noms de fichiers explicites |
| 14 | Balisage Schema | FAQPage, Product, Article — améliore l'affichage SERP |

### Section 3 — Checklist contenu (4 points)

| # | Tâche | Note |
|---|---|---|
| 15 | Stratégie de contenu | Questions clients → mots-clés → types de pages |
| 16 | Format lisibilité | Sommaire ancré, H2/H3 structurés, puces, multimédia, FAQ |
| 17 | Contenu dupliqué | Descriptions originales (pas fabricant) ; rel=canonical si nécessaire |
| 18 | Pages de destination dédiées | 1 URL = 1 sujet en profondeur ; cluster = backlinks + articles internes |

### Section 4 — Checklist technique (8 points)

| # | Tâche | Note |
|---|---|---|
| 19 | Mobile-friendly | Test Google ; thèmes Shopify = responsive natifs |
| 20 | HTTPS | Signal de classement depuis 2014 |
| 21 | Vitesse (CWV) | LCP ≤ 2,5 s ; FID ≤ 100 ms ; CLS ≤ 0,1 — PageSpeed Insights |
| 22 | Liens cassés | GSC → Indexation → Pages ; rediriger les pages supprimées |
| 23 | Maillage interne | Produits connexes, articles croisés, autorité thématique |
| 24 | Texte d'ancrage | Varier : exact / partiel / termes connexes / marque / titre |
| 25 | Navigation globale | Indique aux crawlers les pages prioritaires |
| 26 | Core Web Vitals | Rapport CWV dans GSC + PageSpeed Insights |

### Section 5 — Checklist off-page (4 points)

| # | Tâche |
|---|---|
| 27 | Analyser backlinks concurrents (Moz / Ahrefs / Semrush) |
| 28 | Guest posts : `[niche] + "écrivez pour nous"` |
| 29 | RP digitales : prix, communiqués, HARO, échantillons |
| 30 | Récupérer les mentions sans lien → demander backlink |

### Section 6 — Checklist local (4 points)

| # | Tâche |
|---|---|
| 31 | Créer et optimiser Google Business Profile |
| 32 | S'inscrire sur les 10 principaux annuaires (Facebook, Apple Maps, Yelp, Pages Jaunes…) |
| 33 | Page "Nous contacter" complète (NAP + heures à jour) |
| 34 | Pages de destination par ville si multi-emplacements |

### Bonus 2025 (4 points)

| # | Conseil |
|---|---|
| 35 | IA pour accélérer la production (brouillons, titres, plans) — mais contenu final différencié obligatoire |
| 36 | Diversifier formats selon SERP cible (vidéo, images, texte) |
| 37 | Inviter des experts tiers pour renforcer l'E-E-A-T |
| 38 | UX = facteur de classement indirect (vitesse, mobile, tunnel de commande) |

## Connections / Contradictions vs [[semrush-seo-checklist-41]]

| Shopify (38 pts) | Semrush (41 pts) |
|---|---|
| Orienté débutant e-commerce | Orienté intermédiaire/agence |
| Biais Shopify explicite | Biais Semrush explicite |
| Inclut section Local SEO | Local = 1 seul point (GBP) |
| Bonus IA + UX + formats SERP | Pas de bonus tendances |
| Pas de robots.txt, redirection 302, cannibalisation | Couvre ces points |
| FID encore cité ⚠️ (INP le remplace depuis mars 2024) | Idem |

**Note technique :** Le point 21 cite encore le FID (First Input Delay) parmi les CWV, alors qu'il a été remplacé par l'INP (Interaction to Next Paint) en mars 2024. → Voir [[core-web-vitals]] pour la version à jour.
