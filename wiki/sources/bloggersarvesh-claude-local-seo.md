# Claude + Local SEO 8-Prompt Playbook — Summary

**One-liner:** Sarvesh (14 years local SEO, Alventra Marketing) shares a concrete 8-prompt Claude Code system for full Google Business Profile optimization — category audit, attribute audit, competitor review teardown, review templates, GBP posts calendar, services optimization, description (3 versions), photo plan — with a 4-week execution order → [[google-business-profile]].

## Key takeaways
- Core thesis: "Problem never effort — always execution order." → [[local-seo]].
- **System = Claude Code + pre-loaded business context** (name, address, GBP URL, competitors, target keywords). Without context pre-load, output is generic; with it, every prompt is business-specific.
- **14 years local SEO experience** behind system design (Alventra Marketing).

## The 8-prompt stack

### Prompt 1 — GBP Category Audit
- Search 3 target keywords in Google Maps, collect competitor map pack positions.
- Extract primary + secondary categories from each competitor GBP.
- Spreadsheet: business × categories × ranking position.
- **Highlight: categories competitors have that you don't.**
- Key insight: wrong or missing secondary category = invisible for entire search segments. One added category can trigger new searches within days → [[google-business-profile]].

### Prompt 2 — GBP Attributes Audit
- Extract all visible attributes from own GBP vs. 3 competitors (veteran-owned, free estimates, 24/7, wheelchair accessible, etc.).
- Spreadsheet comparison: mine (yes/no) vs competitors (yes/no).
- **Find: attributes all top competitors share = baseline market requirements.**
- Two-for-one: ranking signal + CTR trust signal before click → [[google-business-profile]].

### Prompt 3 — Competitor Review Teardown
- Read last 50 reviews for each competitor.
- Extract: total count, avg rating, review velocity (last 30/60/90 days), services mentioned, neighborhoods mentioned, recurring complaints.
- Compare velocity vs. own listing.
- **Output: tab with exact reviews/month needed to overtake top competitor + timeline.**
- Key insight: reviews mentioning services + neighborhoods = ranking signal for competitors → [[google-business-profile]].

### Prompt 4 — Review Response Strategy
- Analyze last 30 review responses per competitor (response rate, time, keyword use, negative review handling).
- **Output: template system — 5★, 4★, 3★, 1-2★ templates, 3 variations each** (to avoid robotic appearance).
- Key insight: keyword-rich responses = 120 pieces of keyword-dense GBP content/year at 10 reviews/month → [[google-business-profile]].

### Prompt 5 — GBP Posts Strategy
- Audit competitor post frequency (last 90 days).
- **Output: 8-week posting calendar, 2-3 posts/week** — seasonal promotions, before-after showcases, neighborhood-specific content, review highlights, team spotlights.
- Each post: 1+ target keyword naturally included.
- **Write first 4 weeks of posts (full copy + image suggestions).**
- Key insight: most competitors don't post → consistent posting = immediate differentiation. Neighborhood-specific posts build location relevance hard to replicate → [[local-seo]].

### Prompt 6 — Services Section Optimization
- Extract services + descriptions from own GBP vs. competitors.
- Cross-reference with website: services on site missing from GBP.
- **Output: optimized 2-3 sentence descriptions for all services** (keywords + area + benefit).
- Key insight: services section = only GBP text you fully control. Missing service on GBP = invisible for that search in map pack → [[google-business-profile]].

### Prompt 7 — GBP Description Optimization
- Extract descriptions from own listing + 3 competitors.
- Analyze: character count, keywords, service areas, USPs, CTA.
- **Output: 3 versions (max 750 chars each)**: keyword-focused, conversion-focused, balanced.
- Test versions 30 days each, track impressions + calls → [[google-business-profile]].

### Prompt 8 — GBP Photo Audit
- Count photos (total, last 90 days), categorize types (team, before-after, office, trucks, equipment), flag stock photos.
- **Output: 8-week photo upload plan** — type + quantity per week.
- Google stat: businesses with photos get **42 % more directions requests**, **35 % more click-throughs** → [[google-business-profile]].
- Key insight: consistency > volume. 3-5 quality photos/week signals active business → [[google-business-profile]].

## 4-week execution order
- **Week 1**: Category + Attributes audit (fastest ranking impact, can see changes within days).
- **Week 2**: Services section + Description optimization (fill GBP with indexable keywords).
- **Week 3**: Review teardown + response templates (velocity target + keyword-rich content engine).
- **Week 4**: GBP posts calendar + photo audit (ongoing content system, location relevance compounding).
- **Week 5+**: Execute. Post consistently. Upload photos weekly. Respond to every review.
