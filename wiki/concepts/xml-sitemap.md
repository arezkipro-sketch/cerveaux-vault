# XML Sitemap

**Definition:** A file listing a site's pages so search engines can discover and index them. Usually at `domain.com/sitemap.xml` (or `/sitemap_index.xml`). → [[semrush-submit-sitemap]]

## What we know
- Generate one with a sitemap tool if missing. → [[semrush-submit-sitemap]]
- **Submit** via [[google-search-console]] Sitemaps report — Google can find it alone, but submitting ensures discovery + faster indexing/ranking. → [[semrush-submit-sitemap]]
- Find any site's sitemap via `sitemap site:domain.com filetype:xml`.

## Relations
- A technical-SEO asset feeding [[google-bots]] crawl/[[crawl-budget]]; submitted through [[google-search-console]]; part of [[seo-audit]].
- Complements [[topic-clusters]] (helps engines find all cluster pages).

## Tensions / open questions
- Size limits / split index sitemaps for large sites not deep-dived.
