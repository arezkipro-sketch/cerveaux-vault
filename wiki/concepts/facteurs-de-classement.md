# Facteurs de classement Google (triptyque + méthode Top 3)

**Definition:** Comment Google sélectionne le n°1 : non pas un facteur magique, mais une multitude rangée en **trois catégories cochées dans l'ordre — pertinence → qualité → autorité**. Objectif constant de Google : prédire le résultat qui satisfera le mieux l'internaute (contenu « people-first »).

## Le triptyque (dans l'ordre)
1. **Pertinence** — filtre n°1. La page répond EXACTEMENT à l'intention + sous-intentions. Sans alignement, ni vitesse ni backlinks ne sauvent. Mapping intention→format :
   - Informationnelle (« comment… ») → guide/tuto/étapes/visuels.
   - Transactionnelle (« acheter… », « meilleur… ») → page produit/catégorie, comparatif, avis, réassurance.
   - Locale (« près de moi », ville) → pack local + pages localisées.
   → [[search-intent]], [[410-gone-premier-google-top3]].
2. **Qualité** — départage souvent ; ≠ « texte long ». = utile, unique, clair, crédible, scannable. Signaux : promesse explicite, structure scannable, réponses aux objections (FAQ), confiance (auteur/date/sources). Incarné par [[e-e-a-t]] + **pyramide inversée** (réponse d'abord). → [[410-gone-premier-google-top3]].
3. **Autorité** — tranche à pertinence/qualité égales. ≠ que des backlinks : liens entrants (qualité>quantité), mentions de marque, historique du domaine, réputation, **maillage interne intelligent**. Pèse d'autant plus que la requête est concurrentielle. → [[off-page-seo]], [[backlinks]], [[maillage-interne]].

## Méthode 5 étapes pour viser le Top 3
1. **Gagner l'intention** — « respirer l'odeur de la SERP » (laurent-bourelly) : analyser mot-clé + synonymes + sous-intentions longue traîne + types de contenus + angle + axe de différenciation. Test du plan montré à un proche.
2. **Contenu crédible** — E-E-A-T montré dans la page (vécu, précision, sources, transparence), pas une simple bio ; auto-scoring → [[e-e-a-t]].
3. **On-page** — title/meta marketing, intention dans les 10 premières lignes, H2 = sous-questions SERP/PAA, formats extractibles, maillage, preuves → [[seo-copywriting]].
4. **Technique & UX** — indexation propre, Core Web Vitals, mobile-first, images légères, 404/500 maîtrisés → [[core-web-vitals]], [[mobile-seo]], [[http-status-codes-seo]].
5. **Autorité & backlinks** — autorité interne d'abord (liens depuis pages fortes, ancres variées, cluster), puis backlinks qualité/pertinence/régularité (RP, guest posts, partenariats, mentions, linkable assets) → [[backlinks]], [[maillage-interne]].

## Requêtes concurrentielles
- L'autorité pèse beaucoup plus ; progresser **par paliers : top 10 → 5 → 3 → 1** ; découper en sous-requêtes longue traîne + construire un cluster + accumuler signaux (liens, mentions, partenariats). → [[topical-authority]], [[long-tail-keywords]].
