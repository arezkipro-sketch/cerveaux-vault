# Visual Copywriting

**Definition:** Pairing words with images, colour and layout to capture attention and guide the reader to action — in a saturated feed, text alone no longer converts. → [[academie-copywriting-visuel]]

## What we know
- **Why:** the brain processes an image far faster than text *(≈60,000×, unsourced)*; visuals lift memorisation + engagement. → [[academie-copywriting-visuel]]
- **4 fundamentals:** visual hierarchy (titles/subheads/bullets/contrast guide the eye) · image choice (authentic, supports an argument, not generic stock) · layout (spacing/alignment/responsive) · graphic consistency (colours/fonts → trust). → [[academie-copywriting-visuel]]
- **Mistakes:** generic images, visual overload, unoptimised image weight (hurts SEO), non-responsive. → [[academie-copywriting-visuel]]
- **× SEO:** alt tags + light images + hierarchical HTML + better UX = positive Google signals. → [[academie-copywriting-visuel]]

## Relations
- The visual layer over [[landing-page]], [[email-copywriting]], [[article-structure]]; uses [[color-psychology]] (contrast/CTA) and [[call-to-action]].
- Image-weight/alt overlaps the SEO cluster [[image-seo]] — a copy↔SEO bridge.
