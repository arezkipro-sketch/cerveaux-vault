# Exact Match Domain (EMD)

**Definition:** A domain name that exactly matches a target keyword or phrase (e.g. `taxis-paris.fr` for the query "taxis Paris"). It signals to search engines the site's core topic and can improve relevance scoring for that keyword.

## What we know
- Provides a **keyword signal** to search engines about the site's main topic → [[jotaro-seo-exact-match-domain]].
- **Memorability**: users remember keyword-based domains more easily, supporting return visits → [[jotaro-seo-exact-match-domain]].
- Particularly useful for **niche targeting** and e-commerce, where specificity beats broad positioning → [[jotaro-seo-exact-match-domain]].
- **Best practices**: max 2 keywords; keep domain short and readable; check the keyword's SERP competition; use hyphens between multiple words → [[jotaro-seo-exact-match-domain]].
- Power **declined after Google Penguin** (not dated in source) but EMDs remain valuable when combined with quality content, UX, internal linking ([[maillage-interne]]), and linkbaiting → [[jotaro-seo-exact-match-domain]].
- Brand authority can override EMD advantage — G7 Taxi ranks without an EMD due to brand longevity → [[jotaro-seo-exact-match-domain]].

## Relations
- Complements [[long-tail-keywords]] and [[cocon-semantique]] as part of a full SEO architecture.
- Relates to [[link-juice]] — a well-known EMD may attract more natural links.
