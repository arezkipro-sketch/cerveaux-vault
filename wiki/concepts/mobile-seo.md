# Mobile SEO

**Definition:** Optimising a site to rank and read well on mobile devices. With ~62% of global traffic on mobile (2025), it's a core part of any SEO strategy, not an add-on. → [[semrush-mobile-seo]]

## What we know
- **Why:** ~62% of web traffic is mobile; ~90% of consumers research products on phones, even in-store. → [[semrush-mobile-seo]]
- **Responsive design = Google's recommended setup:** one URL + HTML, CSS adapts per device → single shareable URL, less maintenance, no redirect (faster), Googlebot crawls once, no duplicate content. → [[semrush-mobile-seo]]
- Underpins mobile-first indexing (Google indexes the mobile version).

## Relations
- Part of technical SEO ([[seo-audit]]); the rendering substrate for mobile-first [[call-to-action]], [[landing-page]] and [[visual-copywriting]].
