# Copywriting Tools

**Definition:** Free/low-cost tools that speed up and sharpen copy — editing, ideation, headline scoring, organisation, measurement. Tools amplify but don't replace persuasion technique. → [[academie-outils-gratuits-copywriting]]

## What we know
- **Editing/clarity:** Grammarly, Hemingway App, Google Docs (+ collaboration). → [[academie-outils-gratuits-copywriting]]
- **Ideation:** AnswerThePublic (questions/angles), Thesaurus.com (synonyms). → [[academie-outils-gratuits-copywriting]]
- **Headlines:** CoSchedule Headline Analyzer (emotion/power/clarity score). → [[academie-outils-gratuits-copywriting]]
- **Organise:** Evernote, Notion. **Visuals:** Canva. **Measure:** Google Analytics (CTR/time/conversions). → [[academie-outils-gratuits-copywriting]]
