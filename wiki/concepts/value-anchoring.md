# Value Anchoring

**Definition:** The anchoring bias applied to copy — the first information the reader sees becomes the reference point that frames everything after (value, price, credibility). If you don't set the anchor, the prospect sets an unfavourable one. → [[academie-ancrage-mental]]

## What we know
- **Rule #1:** pilot the frame, or the prospect anchors badly ("just text", "expensive"). You change perception, not reality. → [[academie-ancrage-mental]]
- **5 anchor types:** contrast (before/after) · categorisation/repositioning ("un texte" → "un actif de conversion") · cost of inaction · proof (validated anchor) · framing (same reality, more useful angle). → [[academie-ancrage-mental]]
- **Sales-page sequence:** strong anchor at open → anchor on transformation not features → reinforce with proof → **reveal price only after framing value**. Order: **valeur → preuves → mécanisme → prix**. → [[academie-ancrage-mental]]
- **5 mistakes:** anchor too far from reality, too many anchors, anchor without proof, price too early, vague framing. → [[academie-ancrage-mental]]

## Relations
- An advanced lever of [[copywriting-psychology]]; pairs with [[social-proof]] (proof anchor), [[storytelling]] (contrast), [[scarcity-urgency]] (cost of inaction).
- "Price after value" is a [[sales-funnel]] / sales-page sequencing rule; opens via [[copywriting-frameworks]] / [[hook]].
