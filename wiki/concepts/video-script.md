# Video Script (persuasive)

**Definition:** A text built to guide a viewer to a decision. Most videos fail to convert because the *script* is weak, not the editing. → [[academie-script-video]]

## What we know
- **5 goals:** captivate fast, connect ("c'est moi"), install one simple central idea, prove, trigger action. → [[academie-script-video]]
- **Pre-work:** one action + one problem + one benefit (never multiple); know the audience + use the mirror effect. → [[academie-script-video]]
- **6-part timed structure ("90-sec" template):** Hook (0-3s) → Identification (3-10s) → Central idea (10-20s) → Demonstration/proof (20-60s) → Transition (60-75s) → CTA (75-90s). → [[academie-script-video]]
- **7 hook types:** question, objection, shock phrase, promise, contrast, mistake, curiosity. → [[academie-script-video]]
- **Channel adaptation:** Reels/TikTok (aggressive hook, 1 idea), YouTube (narrative), sales page (proof/objections), ad (ultra-short). → [[academie-script-video]]
- **Mistakes:** product-focused not viewer, no structure, too many ideas, no proof, weak CTA. → [[academie-script-video]]

## Relations
- A video adaptation of [[sales-page-structure]] / AIDA; assembles [[hook]], [[mirror-effect]], [[value-anchoring]] (central idea), [[social-proof]], [[call-to-action]].
- Step 1 of the [[video-marketing]] production process (the script before camera/editing/hosting).
