# SEO Copywriting

**Definition:** Writing that serves **both people and search engines** — content that ranks for target keywords *and* persuades the reader to act. The overlap of [[copywriting]] (persuasion/conversion) and SEO (visibility) → [[seo-copywriting]].

## SEO copywriting vs content writing
| | SEO Copywriting | Content Writing |
|---|---|---|
| Goal | Conversion + visibility | Inform / engage |
| SEO focus | Strategic | Variable |
| CTAs | Essential | Optional |
| Metrics | CTR, conversion, ROI | Time on page, shares |
| Business impact | Direct | Indirect |
→ "Content writing builds trust; SEO copywriting turns trust into measurable results." → [[seo-copywriting]]

## Ranking/conversion elements
- Site speed (<2s, → [[core-web-vitals]]), title tag ([[headline]]), strong content, meta description (CTR not ranking), keyword frequency, page links → [[seo-copywriting]].

## 8-step workflow
1. Ideas (Ubersuggest Content Ideas, AI). 2. Ideas→keywords (Google Trends, AnswerThePublic, Exploding Topics). 3. Optimize title (50-60c)/meta (<160c)/one H1 (20-70c). 4. Complete content, fresh, one problem — **AI as collaborator not replacement**. 5. Keyword frequency/placement, no stuffing. 6. Internal links + optimized anchor ([[maillage-interne]]). 7. Quality external links ([[backlinks]]). 8. Rich media. → [[seo-copywriting]]

## Aligns to search intent
- Match message + structure + CTA to the dominant [[search-intent]] (informational / navigational / commercial / transactional) → [[seo-copywriting]].

## Application e-commerce : fiche produit
- Déclinaison e-commerce de la rédaction SEO → [[fiche-produit-seo]] : `<title>`/`<meta>` par patterns à variables, `<h1>` = référence produit, champ lexical utilisateur, contenu unique anti-duplicate → [[benjamin-thiers-fiches-produits]], [[noiise-fiches-produits-seo]].

## Benchmarks CTR (données Sédestral)

| Élément | Longueur | Impact CTR estimé |
|---|---|---|
| Title tag | 50-60 chars, mot-clé en début | +30 à +50 % |
| Meta description | 155-160 chars, CTA inclus | +15 à +25 % |
| URL | Courte, descriptive | +5 % |

- CTR < 2 % une semaine après publication → retravailler title/meta → [[sedestral-blog-article-seo-2026]]
- Méthode **PAS** pour l'introduction : Problème → Agitation → Solution + mot-clé dans les 100 premiers mots
- Densité mot-clé cible : **1-2 %** = ~15-30 occurrences pour 1 500 mots
- **Quick win** : cibler positions 11-20 avant d'attaquer le top 10
