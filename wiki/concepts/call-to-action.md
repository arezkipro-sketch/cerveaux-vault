# Call to Action (CTA)

**Definition:** The copy that tells the reader exactly what action to take next. Always a command starting with a verb. It is the end of a psychological path and a primary conversion lever. → [[frw-what-is-cta]], [[academie-cta-2025]]

## What we know
- **Always a verb/command:** "Buy now", "Sign up", "Réservez votre place". Tell them *exactly* what to do; don't make them think about the step or the benefit. → [[frw-what-is-cta]]
- **Funnel stage sets strength:** top-of-funnel → *soft* ("Learn more"); bottom-of-funnel → *direct* ("Get started", "Buy now"). → [[frw-what-is-cta]]
- **Psychological levers:** benefit anticipation, risk reduction ("Essai gratuit", "Sans engagement", "Annulable"), authentic urgency ("Plus que 3 places"). Excess = distrust. → [[academie-cta-2025]]
- **Wording 2025:** precise + personalised beats generic; first-person ("Oui, je veux…") often beats "En savoir plus"; "Cliquez ici"/"Click here" is antiquated (users tap/swipe/voice; "click" isn't the real action). → [[academie-cta-2025]], [[frw-what-is-cta]]
- **Medium:** digital → linked button/text (buttons can lift CTR, but test); print → short memorable URL. → [[frw-what-is-cta]]
- **Placement:** visible, repeated strategically (top / after offer / conclusion); never buried low; mobile = thumb-clickable. Stands as its own line in the copy doc. → [[academie-cta-2025]], [[frw-what-is-cta]]
- **Always test:** wording, urgency words, length, POV, colour — design held constant for a true A/B. A verb change can 2× conversions. → both
