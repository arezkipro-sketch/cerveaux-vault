# Marketing Tools Landscape

**Thesis:** A catalogue of tool categories a marketer/copywriter draws on, aggregated from Semrush "best tools" listicles (all commercially biased toward Semrush — treat rankings as directional). Pick by job-to-be-done, not the list order.

## Categories (with source listicles)
- **Keyword research:** Semrush Keyword Magic, Google Keyword Planner, AnswerThePublic, etc. → [[semrush-keyword-research-tools]]. See concept [[keyword-research]], [[copywriting-tools]].
- **Content creation:** AI writers, video/photo editors, design (Canva), grammar (Grammarly/Hemingway). → [[semrush-content-creation-tools]]. See [[copywriting-tools]], [[ai-assisted-copywriting]], [[visual-copywriting]].
- **Social media analytics:** track your KPIs, performance, audience across networks. → [[semrush-social-analytics-tools]]. See [[competitor-analysis]].
- **Site performance:** speed testing, audits, monitoring, CDN/caching (Core Web Vitals). → [[semrush-website-performance-tools]]. See [[site-health]], [[mobile-seo]].
- **Social monitoring (listening):** track mentions/conversations/sentiment. → [[semrush-social-monitoring-tools]]. See [[competitor-analysis]].
- **Online reputation management:** monitor + respond to reviews across platforms. → [[semrush-orm-tools]]. See [[online-reputation-management]].
- **Competitive intelligence:** reveal competitors' traffic/keywords/ads/positioning. → [[semrush-competitive-intelligence-tools]]. See [[competitor-analysis]].
- **Advertising:** audience research, ad creation, campaign management. → [[semrush-advertising-tools]]. See [[sem]], [[display-advertising]].
- **AI-SEO tools:** AI-powered content optimisation, keyword/topic research, briefs. → [[semrush-ai-seo-tools]]. See [[ai-seo]], [[geo-tools]].
