# Pinterest

**Definition:** A **visual search engine** (not just social) — ideal for retail, decor, fashion, and lifestyle → [[search-everywhere-optimization]].

## What we know
- Optimize: pin titles + descriptions, board names, visual quality, **alt tags**, keyword-rich bio; consistency is key → [[search-everywhere-optimization]].

## Paid side
- **[[pinterest-ads]]:** inspiration-based advertising reaching users early in the journey; long-lived pins; intention targeting; strong ROAS for considered/visual purchases → [[pinterest-ads]].

## Pinterest Organique e-commerce

⚠️ **Conflit de sources — volume d'épingles/jour :**
| Source | Volume recommandé |
|---|---|
| [[guide-dropshipping-seo-halal]] | 20-30 épingles/jour |
| [[ebook-business-halal]] | 5-15 épingles/jour |

Les deux sources viennent du même auteur (Mimine / mimine-ecom-seo), publiées à des dates différentes. Le consensus : la régularité prime sur le volume brut.

**Données plateforme** (via [[ebook-business-halal]]) : 454 millions d'utilisateurs actifs/mois, 60% de femmes, croissance +9%/an. Meilleures niches : décoration, vêtements, jardinage, mode.

**Paramètres communs aux deux sources :**
- **Structure** : 1 board = 1 collection produit du catalogue
- **Descriptions** : uniques entre épingles (pas de duplication)
- **Mots-clés** : dans le titre de l'épingle + description + nom du board
- **Rich Pins** : activer pour synchroniser prix + disponibilité automatiquement
- **Timing** : pics d'engagement 19h-22h (via [[guide-dropshipping-seo-halal]])
- **Signal prédictif** : Pinterest Trends anticipe de 2-4 semaines les tendances Google ([[jotaro-referencement-predictif-google-trends]])
- **Extension** : "Bouton Enregistrer Pinterest" (6M utilisateurs) pour faciliter les épingles depuis le site

**Process 9 étapes** (via [[ebook-business-halal]]) : images (recto/verso + variantes) → compte pro → réutiliser mots-clés site → nommer boards avec kws collections → descriptions uniques → optimiser site → extension Bouton Enregistrer → profil actif → connecter Shopify
